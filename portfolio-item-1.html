Bay Area NFT Digital Art NFT Coffee Table book

Buy and Stake NFT's to be used in physical Art Book

Summary: My current personal project in the blockchain space. This is a working NFT sale site. The flyer images have been minted and stored on IPFS and the ERC-721 contract live on the kovan test network. Connect with meta mask on the kovan network and you can purchase an NFT with test ether. The nft then can be added to your metamask wallet by adding the NFT contract address.

The long term goal is to create a real world coffee table art book of NFTs sold on the platform. NFT holders will have the ability to stake their NFTs on the protocol and earn the protocols governance token. NFT holders who stake their NFT in the platform will also earn percantages of sales from the coffee table book. All NFTs minted and sold on site will be in the book. All book sales will go back to the protocol and redistributed to token holders.

The goal is to bridge the gap between digital art and physical art, proving a strong usecase for digital NFTs besides speculative value.

Truffle: The framework used to interact with the solidity contracts and the test blockchains.

Ganache: a local blockchain used to replicate the etherum network to help develop locally.

Soldity: the smart-contract language used to perform functions on the blockchain. The open-zeppelin library was used for the ERC-721 standard

Web3JS: web3JS library is the gate way toconnect the browser and metamask wallet to the etherum blockchain

REACTJS: is the Javascript library used to create frontend components.

What I learned: This is my third blockchain project and am now getting used to the tools and flow to create and interact with the etherum network. A lot of the technologies are new and evolving fast so it has really forced me to research deep on issues I run into. I have been able to use the skills i've picked up in the Springboard bootcamp to build out the front end, and have learned to adapt other backend skills to the blockchain.

I am excited to work on some of the features mentioned above. Because the space is still in an experimental phase it should be a great oppurtunity to explore and learn.
